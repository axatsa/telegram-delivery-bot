# Telegram Delivery Bot

Aiogram-based Telegram bot for managing delivery orders — customers browse catalog, place orders, and admins track and fulfill them.

## Features
- Product catalog with inline navigation
- Shopping cart and order placement
- Order status tracking
- Admin panel for order management
- Image support for product listings
- User registration flow

## Tech Stack
- Python 3.11+
- aiogram 3.x
- SQLite

## Setup
```bash
pip install -r requirements.txt
# Add BOT_TOKEN in main.py
python main.py
```
